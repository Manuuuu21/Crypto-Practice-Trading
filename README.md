# 🚀 Crypto Practice Trading

A cryptocurrency trading simulator for practicing spot and futures trading with realistic candlestick charts.

**[Try it Live](https://manuuuu21.github.io/Crypto-Practice-Trading/)**

## Features

- **Spot & Futures Trading**: Buy/sell or trade with leverage (1x - 125x)
- **Real-time Charts**: 5s, 15s, 30s candlesticks with MA(7) and Bollinger Bands
- **Drawing Tools**: Trendlines and rectangles for technical analysis
- **Order Book**: Live bid/ask visualization
- **Position Tracking**: Monitor P&L, liquidation price, and portfolio stats
- **Real BTC Prices**: Live data from CoinGecko API

## How to Use

1. Start with 1,000 USDT practice balance
2. Choose **Spot** or **Futures** mode
3. Set price and amount (or use the % slider)
4. Click **Buy/Sell** to execute trades
5. For **Futures**: Select leverage and open Long/Short positions
